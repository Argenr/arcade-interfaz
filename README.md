# Interfaz

Proyecto personal en desarrollo de una interfaz gráfica para launcher de juegos, creada en Python con PySide6.

El objetivo es experimentar con:
- navegación por teclado
- diseño visual tipo arcade/consola
- organización y lanzamiento de juegos desde una interfaz propia

Actualmente el proyecto **está en construcción**.

---

## Estado del proyecto
En desarrollo activo.  
La estructura, animaciones y navegación están siendo iteradas constantemente.

---

## Funcionalidades actuales
- Interfaz gráfica en Qt (PySide6)
- Navegación en grid mediante teclado
- Sistema de foco visual con animaciones
- Scroll automático al cambiar de selección
- Carga de información desde archivo `games.json`
- Estilos personalizados con QSS

---

## Tecnologías utilizadas
- Python 3
- PySide6 (Qt)
- Qt Style Sheets (QSS)

---

## Ejecución
```python main.py```
